목표:
미국주식 투자자를 위한 한국어 기반 금융 터미널을 만들고 싶어. 미국 주식 데이터, 미국 뉴스, SEC 공시, 실적, 옵션, ETF, 환율, 금리, 원자재 데이터를 중심으로 하고, 한국 주식과 DART 공시도 함께 지원해줘.
필수 기능:
1. 실시간 또는 근실시간 시장 데이터
- 미국 주식, ETF, 지수, 금리, 원자재, 환율, 한국 주식 지원
- API 키가 필요한 경우 어떤 API가 필요한지 문서화
- 데이터가 없으면 가짜 숫자로 채우지 말고 “데이터 없음 / API 필요 / 지연 데이터”처럼 명확히 표시
- 비로그인 사용자도 일반 시장 정보는 실제 데이터로 볼 수 있게 해줘
2. 뉴스와 번역
- 미국 주식 뉴스 원문과 한국어 번역을 함께 표시
- 뉴스 감성, 관련 티커, 중요도, 요약 제공
- Gemini API 또는 다른 AI API를 선택적으로 연결할 수 있게 해줘
3. AI 어시스턴트
- Gemini API를 선택해서 사용할 수 있게 해줘
- 종목, 뉴스, 공시, 차트, 포트폴리오를 바탕으로 한국어 요약/분석 제공
- API 키가 없으면 로컬 규칙 기반 요약이라도 작동하게 해줘
4. 차트
- 캔들 차트, 거래량, 이동평균, Bollinger Band, RSI, MACD
- 1M / 3M / 6M / 1Y / 2Y / 5Y / 10Y 기간 변경
- 1D / 1W / 1M 인터벌 변경
- 기간과 인터벌 변경 시 실제로 차트 데이터가 바뀌어야 함
- 포트폴리오 그래프는 크게 보기 모드 제공
5. 포트폴리오
- 로그인하면 내 실제 포트폴리오를 볼 수 있게 해줘
- 브로커 API 연동 구조를 만들어줘. 예: Alpaca, Interactive Brokers, 한국투자증권 Open API 등
- 보유 종목, 평가금액, 손익, 비중, 섹터 비중, 국가 비중, 통화 비중, 배당 예상, 리밸런싱 필요 여부 표시
- 사용자가 직접 포트폴리오를 수동 입력/수정할 수도 있게 해줘
6. 로그인과 개인 설정
- 비로그인: 일반 시장 데이터와 뉴스 조회 가능
- 로그인: 내 포트폴리오, 관심종목, API 키, AI 설정, 레이아웃, 알림, 기본 종목 저장 가능
- 사용자별 설정 저장
- 민감한 API 키 저장 방식과 보안 주의사항 문서화
7. UI/UX
- 첨부 이미지처럼 전역 메뉴를 고밀도 금융 터미널 스타일로 구성
- 상단에는 전역 메뉴, 명령창, AI 버튼, 지수 스트립 배치
- 메뉴 예시: Markets, Portfolio, Research, Tools, AI
- 내부 탭 예시: 시장, 모니터, 차트, 뉴스, 포트폴리오, 옵션, 주문, AI
- 각 탭은 실제로 화면이 전환되어야 함
- 각 항목 박스는 드래그앤드롭으로 위치 변경 가능
- 박스 크기도 사용자가 늘리고 줄일 수 있게 해줘
- 좌측/중앙/우측 패널 경계도 드래그로 폭 조절 가능
- 레이아웃은 사용자별로 저장되게 해줘
- 화면이 잘리지 않도록 패널 내부 스크롤과 반응형 레이아웃을 잘 설계해줘
8. 주문/거래
- 실제 주문 기능은 안전하게 분리
- 기본은 Paper Trading으로 제공
- 실제 브로커 주문 연동은 설정에서 명확히 켜야만 가능하게 설계
- 실제 매매와 모의 매매가 절대 혼동되지 않도록 UI에 표시
9. 배포
- 로컬 실행 가능
- 실서버 배포 가능
- Docker / docker-compose 제공
- 환경변수 예시 파일 제공
- HTTPS, 로그인, 세션, API 키 보안 관련 문서 제공
- 운영 서버에 올리는 방법을 문서화

품질 기준:
- 첫 화면 로딩이 느리면 안 됨
- 초기에는 빠른 시장 데이터부터 표시하고, 뉴스/공시/AI/포트폴리오는 백그라운드로 불러와줘
- 데이터가 없을 때 가짜 숫자를 실제처럼 표시하지 마
- 실제 데이터, 지연 데이터, API 제한, 데이터 없음 상태를 명확히 구분해줘
- 최종 ZIP을 주기 전에 직접 실행 테스트하고 오류를 고쳐줘
